# Artisan Bakery Website
 responsive HTML website , showcasing a menu, story, and contact information with smooth scrolling navigation.

#Features
-Responsive Design: Adapts to different screen sizes.

-Navigation: Smooth scrolling to sections (Home, About, Menu, Contact).

-Menu Section: Displays featured items with images, prices, and "Add to Cart" buttons.

-About Section: Includes bakery story and image.

-CTA Section: Call-to-action for ordering with multiple options.

-Footer: Contact details, hours, social links, and copyright.

##How to View
-Download or clone the repository.
-Open index.html in any modern web browser.
-No server required; it's a static HTML page.

##Technologies Used
-HTML5: Structure and content.
-JavaScript: Smooth scrolling functionality.

##File Structure
-index.html: Main website file.

##External links: Fonts from Google Fonts, icons from Font Awesome.
